# iToolBox-Extension

iToolBox 扩展仓库，所有的扩展都在这里边。

## 插件规划

- [x] API：能力API DEMO
- [x] Url：url操作扩展
- [x] QRCode：二维码扩展
- [ ] IMG2Base64：图片转base64扩展
- [ ] JS PlayGround：JavaScript 运行平台
- [ ] Regular：正则表达式查询扩展
- [ ] Color：颜色转换，主题色生成
- [ ] Scale：进制转换
- [ ] NPM：npm 仓库搜索，仓库推荐
- [ ] TextCompare：文本比较
- [ ] PinYin：文字转拼音，文字加拼音
- [ ] WechatJS：微信JS签名生成校验工具
- [ ] WechatCard：微信卡劵签名生成校验工具
- [ ] CodeTerser：代码压缩
- [ ] GitHub：GitHub用户查询，仓库查询
- [ ] JSON：json格式化
- [ ] QiniuCDN：七牛cdn上传文件
